# 💊 医药代表培训多智能体系统 (MTS)

基于多智能体架构的个性化医药代表培训与科室主任拜访辅助系统。

## 四大智能体

| 智能体 | 职责 | 说明 |
|--------|------|------|
| **文档分析专家** | 文档解析与理解 | 支持 PDF/DOCX/TXT/MD/PPTX 上传，自动提取药品信息、竞争优势、临床数据等 |
| **培训出题专家** | 生成培训问题 | 生成多维度（产品知识/竞品分析/临床场景/销售技巧/合规）、多难度的培训题 |
| **科室主任（角色扮演）** | 模拟拜访对话 | 预设多种主任性格模板，支持自定义角色，动态调整对话态度 |
| **培训评估专家** | 评分与反馈 | 六维度评分体系，逐轮点评，提供推荐话术和培训建议 |

## 快速开始

### 1. 环境准备

```bash
# 创建虚拟环境
python -m venv .venv
.venv\Scripts\activate       # Windows
# source .venv/bin/activate  # Mac/Linux

# 安装依赖
pip install -r requirements.txt
```

### 2. 配置 API Key

复制 `.env.example` 为 `.env`，填入 API Key：

```bash
copy .env.example .env
```

编辑 `.env` 文件：

```env
OPENAI_API_KEY=sk-your-key-here
OPENAI_MODEL=gpt-4o
```

如使用 Azure OpenAI，修改 `LLM_PROVIDER=azure` 并填写 Azure 相关配置。

### 3. 启动系统

```bash
streamlit run app.py
```

浏览器将自动打开 `http://localhost:8501`。

## 使用流程

### Step 1: 上传文档
进入「📁 文档管理」，上传产品手册、销售策略等文件。系统自动解析并生成分析报告。

### Step 2: 生成培训题目
进入「📝 培训题库」，选择题目数量、难度和领域，一键生成培训题。也可描述特定场景生成情景化题目。

### Step 3: 角色扮演训练
进入「🎭 角色扮演」，选择或自定义科室主任角色，开始模拟拜访对话。

### Step 4: 获取评估报告
- 角色扮演结束后点击「评估本次拜访」获取全面评估
- 在「📊 问答评估」中单独评估问答对或批量评估

## 评估维度

系统采用六维度评分体系（每项满分 10 分）：

1. **产品知识准确性** - 信息是否准确完整
2. **临床证据引用** - 是否有循证医学支持
3. **沟通技巧** - 表达是否清晰有条理
4. **异议处理能力** - 面对质疑的应对能力
5. **合规性** - 是否在合规框架内推广
6. **价值传递** - 是否有效传递产品价值

总体评级：优秀(≥85%) / 良好(≥70%) / 合格(≥60%) / 需改进(<60%)

## 技术栈

- **向量存储**: FAISS + LangChain
- **前端**: Streamlit
- **文档解析**: PyPDF2 / python-docx / python-pptx
